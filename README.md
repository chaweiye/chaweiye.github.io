> 「 星河滚烫，你是人间理想 」

## 🎨 设计理念

### 主题方向
- **神秘感**：深色星空背景 + 粒子动效 + 流星划过
- **高级感**：紫粉渐变配色 + 毛玻璃效果 + 细腻动画
- **温情知性**：诗意文案 + 优雅字体 + 人文气息

### 技术特性
✅ **纯静态页面** - GitHub Pages 原生支持，无需构建  
✅ **响应式设计** - 完美适配桌面/平板/手机  
✅ **CSS3 动画** - 星空闪烁、流星划过、呼吸光环  
✅ **滚动交互** - 导航栏变化、元素淡入、平滑滚动  
✅ **Google Fonts** - 中文字体优化（Noto Serif SC + 站酷小薇）

## 🚀 部署到 GitHub Pages

### 方法一：直接替换
1. 将本项目所有文件复制到你的 `chaweiye.github.io` 仓库
2. 覆盖原有 `index.html`、`css/`、`js/` 目录
3. 提交并推送，GitHub Pages 自动部署

### 方法二：本地预览
```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8000

# 浏览器访问 http://localhost:8000
```

## ✨ 功能亮点

### 🌌 视觉效果
- **动态星空**：150+ 星星粒子随机闪烁
- **流星划过**：定时随机生成流星动画
- **头像光环**：呼吸渐变 + 旋转光环
- **毛玻璃卡片**：backdrop-filter 半透明效果
- **渐变文字**：紫粉渐变文字质感

### 🎯 交互体验
- **智能导航**：滚动时自动高亮当前区块
- **平滑滚动**：锚点链接平滑过渡
- **悬停动效**：卡片上浮、标签缩放
- **滚动淡入**：元素进入视口时淡入动画
- **导航毛玻璃**：滚动时背景模糊效果

### 📱 响应式布局
- 桌面端：网格布局，多列展示
- 平板端：自适应列数
- 移动端：单列布局，汉堡菜单（可扩展）
